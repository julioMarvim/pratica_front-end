Tamanho proporcional ao tamanho do container pai.

Se aplicarmos uma class incluindo uma formatação específica para um elemento filho, ela tem ordem de prioridade em relação a formatação do container pai. Neste caso temos o container pai com um tamanho fixo de 30px, porem essa tag "p" possui uma formatação para tamanho relativo de 100%.
